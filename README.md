## 介绍

个人主页，Vue3.2+Vite+TypeScript+Pinia+Setup 写法

## 特性

- 有专门用来练习 Vue3.2 的 TodoList 案例
- 可以快速入门 Vue3.2 + TypeScript
- 有 Vue3 + Setup 组件写法
- 有 Vue3.2 命令式组件写法
- 有 jest + TypeScript 入门写法，写出第一个 Vue 组件测试用例，查看覆盖率

## 用法

1.安装 yarn，如果已经安装 yarn，此步骤跳过

```
npm i -g yarn
```

2.安装依赖

```
yarn
```

3.运行项目

```
yarn dev
```
