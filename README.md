## 介绍

个人主页，Vue3.2+Vite+TypeScript+Pinia+Setup 写法，有专门用来练习 Vue3.2 的 TodoList 案例，快速入门 Vue3.2+TypeScript。

## 用法

1.安装 yarn，如果已经安装 yarn，此步骤跳过

```
npm i -g yarn
```

2.安装依赖

```
yarn
```

3.运行项目

```
yarn dev
```
